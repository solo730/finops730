# ⚡ FinOps Study App

A free, offline-first mobile study app for the **FinOps Certified Practitioner (FOCP)** exam. Built by **FinOps730 LLC** for anyone looking to learn FinOps — especially those new to cloud.

No login. No internet required. Just save the file and open it in any mobile browser.

---

## 📱 Features

### 📚 Flashcards (70 cards)
- Covers all major exam topic areas
- Tap to flip between question and answer
- Filter by topic section
- Shuffle mode for randomized practice
- Progress bar tracking

### 🎯 Quiz (35 questions)
- Multiple choice with instant feedback
- Color-coded correct / incorrect answers
- Full explanation after every question written in plain language
- Score screen with personalized feedback
- Best score saved locally on your device
- Questions randomized every session

### 📖 Glossary (55+ terms)
- Complete A–Z FinOps terminology reference
- Live search bar — search by term, definition, or category
- Tap any term to expand its definition
- Category tags for quick reference (Core Concept, Finance, Capability, FOCUS Metric, etc.)

## 🚀 How to Use

### On Mobile (Recommended)
1. Download `FinOps_Study_App.html`
2. Save it to your phone
3. Open it in Chrome (Android) or Safari (iOS)
4. Works 100% offline — no internet needed after download

### On Desktop
1. Download `FinOps_Study_App.html`
2. Open it in any web browser
3. Works immediately — no install, no dependencies

### Study Strategy
1. **Flashcards first** — go through all cards at least twice
2. **Use the Glossary** — look up any unfamiliar terms
3. **Take the Quiz** — aim for 75%+ (that's the real exam passing score)
4. **Repeat** — shuffle the cards and retake the quiz until you're consistent
